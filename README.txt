# How to Update Your Website

1. **Create a Folder**: 
   Inside the `news` folder, create a new folder with the format `Date(Number)`.
   Example:
   - First news of the day: `2026-02-18(1)`
   - Second news of the day: `2026-02-18(2)`
   - Third news: `2026-02-18(3)`

2. **Add Content**:
   - Create a text file (`details.txt`) inside that folder. 
     - **Line 1**: Title
     - **Line 2+**: News Description
   - Add an image file (jpg, png) inside the folder.

3. **Update the Site**:
   - Double-click `RUN_UPDATE.bat`.

4. **View**:
   - Open `index.html`.
